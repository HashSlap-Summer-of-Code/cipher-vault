# 🔒 Secure File Vault

A secure file vault web application built with Node.js and Express that provides AES encryption for uploaded files with user authentication and access control.

## ✨ Features

- **🔐 AES Encryption**: All uploaded files are encrypted using AES-256-GCM before storage
- **🔑 User Authentication**: Session-based authentication with username/password
- **📁 File Management**: Upload, download, and delete encrypted files
- **🛡️ Access Control**: Users can only access their own files
- **🎨 Modern UI**: Clean, responsive interface with drag-and-drop file upload
- **📊 File Metadata**: Display file size, upload date, and original filename
- **🚀 Security**: Encrypted filenames, secure sessions, and input validation

## 🛠️ Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
